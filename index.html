D20 King

Dice Rolls

  • Each side rolls a d20 to determine thier base chance. If a specific skill is used add modifier, otherwise use base value.
  • Natural 20s are automatic critical success. In the case of both sides getting a 20 both get simultaneous critical success.
  • Any value after modifier below 5 is a failure. Any value 3 or below is a fumble. (Drop weapon/fall down)
  • If character has 2hp or less all actions have -2 modifier and movement speed is half. A character with 1 hp cannot move until healed.
  • If a character has 0 or below hp. The charicter is knocked out. Each round they can roll 15 to wake up. Once awake need a 10 to get up with 2hp
